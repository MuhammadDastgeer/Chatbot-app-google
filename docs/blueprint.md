# **App Name**: User Registration Portal

## Core Features:

- User Input Form: A form with fields for username, email, and password.
- Data Submission: Submits the user-entered data to the specified URL (https://o4tdkmt2.rpcl.app/webhook-test/register-user).

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and security in user registration.
- Background color: Very light lavender (#F0F2FA), almost white, creates a clean, uncluttered aesthetic.
- Accent color: Electric Purple (#BF5AF2) to highlight CTAs and important UI elements.
- Body and headline font: 'Inter', a sans-serif font known for its modern and readable design.
- Simple, outline-style icons for form fields (user, email, password) for clarity.
- Clean and straightforward layout with a focus on form fields for easy input.
- Subtle animations for form validation and submission feedback.
